Our Goal

Our goal with this project was to create a user friendly application for our users to be able to transmit their handwritten notes into sharable online content.

Our Solution

Our solution consists of a model whose implementation depends on numpy, cv2 and tenserflow imports. It consists of 5 CNN layers, 2RNN (LSTM) layers and the CTC loss and decoding layer. The model was deployed using a flask app with a html form utilizing the Get and Post method to collect text to be analyzed.

Our Path to Accomplishment!

We accomplished this by deploying a machine learning text recognition model via a flask app interface. Users are able to upload any file of text image and our model analyze and predict the printed text of the image.
